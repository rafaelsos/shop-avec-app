## shop-avec-app

App para testar conhecimentos JavaScript, React e Redux.

# Front-end

O projeto foi desenvolvido em React, Redux, Saga, Redux Persist, TypeScript... Utilize o [CRA (create-react-app)]( https://facebook.github.io/create-react-app/) para realizar o bootstrap do projeto.

O Front-End foi utilizado a lib [Material UI](https://material-ui.com/getting-started/installation/).


# Bibliotecas

`git clone https://github.com/rafaelsos/shop-avec-app.git`

`npm install`

# Back-end

Rodar o comando: `json-server -p 4000 --watch rest-api/products.json`

Isso irá disponibilizar uma api REST rodando no endereço http://localhost:4000/products.

# Rodar projeto

`npm start`
